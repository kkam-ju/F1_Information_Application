🏎️ F1 Information Application (F1 정보 어플리케이션)
C# Windows Forms(WinForms)를 기반으로 제작된 2025 시즌 포뮬러 1(F1) 정보 및 굿즈 스토어 어플리케이션입니다. F1 팬들을 위해 경기 일정, 드라이버 정보 조회, 그리고 팀별 굿즈 구매 시뮬레이션 기능을 제공합니다.

📅 프로젝트 개요 (Project Overview)
이 프로젝트는 F1의 2025년 시즌 정보를 사용자에게 제공하고, 각 팀의 굿즈를 가상으로 구매해볼 수 있는 키오스크 형태의 응용 프로그램입니다.

개발 언어: C#

프레임워크: .NET Windows Forms

개발 도구: Visual Studio

✨ 주요 기능 (Key Features)
1. 2025 시즌 경기 일정 및 결과 조회
ListView를 활용하여 2025년 F1 그랑프리 일정(날짜, 장소)과 경기 결과(우승자, 우승 팀)를 확인할 수 있습니다.

오스트레일리아 GP부터 아부다비 GP까지의 시즌 데이터를 포함하고 있습니다.

2. 드라이버 및 팀 정보 조회
각 팀(맥라렌, 페라리, 메르세데스, 레드불 등) 버튼을 클릭하면 소속 드라이버의 프로필(이름, 국적, 월드 챔피언십 횟수)과 사진을 확인할 수 있습니다.

3. 팀별 굿즈 스토어 (Merchandise Store)
팀(예: 페라리, 윌리엄스)을 선택하여 굿즈(티셔츠, 자켓, 모자 등)를 장바구니에 담고 결제하는 기능을 시뮬레이션합니다.

기능 상세:

상품 선택 시 장바구니 목록(ListBox)에 수량과 함께 추가됩니다.

실시간으로 총 결제 금액과 수량이 계산됩니다.

'결제하기' 및 '초기화' 기능을 제공합니다.

4. 팀 선택 및 네비게이션
메인 화면에서 각 F1 팀의 로고/버튼을 선택하여 해당 팀의 상점이나 정보 페이지로 이동합니다.

일부 팀(레드불, 자우버 등)은 재고 부족 알림을, 일부는 준비 중 알림을 띄우는 등 예외 처리가 되어 있습니다.

📂 소스 코드 구조 (Source Code Structure)
Program.cs: 어플리케이션의 진입점입니다. Form12를 메인 폼으로 실행합니다.

Form12.cs: (추정) 어플리케이션의 시작 화면 또는 메인 허브 역할을 합니다.

Form1.cs (Team Selection): 각 팀별 스토어나 정보 페이지로 이동할 수 있는 메뉴 화면입니다.

Form11.cs (Info Center):

2025 시즌 캘린더와 경기 결과를 보여줍니다.

팀 버튼 클릭 시 드라이버 정보와 이미지를 로드하여 보여줍니다.

Form_Ferrari.cs (Shop Example): 페라리 팀의 굿즈를 구매하는 쇼핑몰 로직이 구현되어 있습니다. 상품 가격 계산 및 장바구니 로직을 포함합니다.

Form_will.cs / Form_mercedes.cs: 각 팀별(윌리엄스, 메르세데스) 전용 페이지입니다.

⚠️ 주의사항 (Note)
현재 코드상에서 이미지 파일 경로가 로컬 절대 경로(C:\김주엽\C# 개인 프로젝트 (F1)\...)로 설정되어 있습니다. 이 프로젝트를 다른 PC에서 실행할 경우, 해당 경로에 이미지가 없으면 오류가 발생할 수 있습니다.

해결 방법: 이미지 파일을 프로젝트 폴더 내(Resource 등)에 포함시키고 경로를 상대 경로로 수정하는 것을 권장합니다.

"서버 준비 중" 또는 "재고 없음"으로 표시된 팀들의 기능 추가 구현.
