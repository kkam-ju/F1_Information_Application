🏎️ F1 Information Application (F1 정보 어플리케이션)
C# Windows Forms(WinForms)를 기반으로 제작된 2025 시즌 포뮬러 1(F1) 정보 조회, 굿즈 스토어, 그리고 경기 티켓 예매를 체험할 수 있는 어플리케이션입니다.

📅 프로젝트 개요 (Project Overview)
이 프로젝트는 F1 팬들을 위해 2025년 시즌 정보를 제공하고, 팀별 굿즈 구매와 그랑프리 티켓 예매 과정을 시뮬레이션할 수 있는 키오스크 형태의 프로그램입니다.

개발 언어: C#

프레임워크: .NET Windows Forms

개발 도구: Visual Studio

✨ 주요 기능 (Key Features)
1. 2025 시즌 경기 일정 및 결과 조회
경기 일정: 2025년 F1 그랑프리 일정(날짜, 장소)을 ListView로 확인할 수 있습니다.

드라이버 정보: 각 팀 로고를 클릭하면 소속 드라이버의 프로필(이름, 국적, 챔피언십 기록)과 사진을 제공합니다.

2. 팀별 굿즈 스토어 (Merchandise Store)
쇼핑 시뮬레이션: 페라리, 윌리엄스 등 각 팀의 굿즈(의류, 모자 등)를 장바구니에 담고 구매하는 과정을 체험할 수 있습니다.

기능: 상품 수량 조절, 장바구니 목록 확인, 총 결제 금액 실시간 계산 기능이 포함되어 있습니다.

3. 티켓 예매 및 결제 시스템 (Ticket Reservation) 🆕
로그인 기능: 티켓 예매는 로그인 후 이용 가능하며, 아이디와 비밀번호 확인 로직이 구현되어 있습니다 (테스트 계정: abcd / 1234).

좌석 선택: VVIP, VIP, 일반석, 자유석 등 등급별 좌석을 선택하면 총 금액이 자동으로 계산됩니다.

결제 프로세스:

포인트 적립: 포인트 적립 동의 여부를 선택할 수 있습니다.

결제 수단: 일시불/할부, 카드사 선택, 포인트 사용 등 다양한 결제 옵션을 제공합니다.

최종 결제: 선택한 정보(좌석, 결제 수단, 금액)를 종합하여 결제를 완료합니다.

📂 소스 코드 구조 (Source Code Structure)
Program.cs: 어플리케이션의 진입점(Entry Point)입니다.

Form12.cs (Main Menu): 정보 조회, 굿즈샵, 티켓 예매 메뉴로 이동하는 메인 허브입니다.

Form11.cs (Information): 경기 일정 및 드라이버 정보를 보여주는 화면입니다.

Form1.cs (Shop Hub): 각 팀별 굿즈 스토어로 연결되는 선택 화면입니다.

Form14.cs (Login): 티켓 예매를 위한 로그인 화면입니다. 지정된 ID/PW로 인증을 수행합니다.

Form13.cs (Ticket & Payment): 좌석 선택, 금액 계산, 결제 수단 선택 등 예매의 전반적인 로직을 담당합니다.

Form_Ferrari.cs, Form_will.cs, Form_mercedes.cs: 각 팀별 개별 스토어 화면입니다.

⚠️ 주의사항 (Note)
이미지 경로: 코드 내 이미지 경로가 로컬 절대 경로(C:\김주엽\...)로 설정되어 있어, 실행 환경에 따라 이미지가 보이지 않을 수 있습니다. 해당 경로를 수정하거나 리소스 폴더를 생성하여 상대 경로로 변경하는 것을 권장합니다.

로그인 정보: Form14.cs에 하드코딩된 아이디(abcd)와 비밀번호(1234)를 사용하여 티켓 예매 기능을 테스트할 수 있습니다.네, 맞습니다. Form13.cs(티켓 예매)와 Form14.cs(로그인) 파일 내용을 확인해보니 티켓 구매와 관련된 중요한 기능이 구현되어 있었네요.
